Individual: Multi-Condition Boolean
We want to examine the relationship between Loan to Income and Debt to Income (DTI) for each individual Member ID across each Grade. 

Create calculation that computes a Loan to Income ratio and compare this to DTI in a scatter plot. This scatter plot should include a separate mark for each Member ID and should be broken down by Grade. Next, create a Boolean calculation that checks to see whether or not members have both low DTI ratios (<20) and low Loan to Income ratios (<15%), and color the visualization using this calculation.


1 Create the [Loan/Income Ratio] calculation.

[Loan Amnt] / [Annual Inc]

2 Create the basic scatter plot.
Drag [Dti] to rows, [Loan/Income] to columns, and [Member ID] to the Detail Marks card. * You may notice some outliers. If so, lasso these 3 points and exclude them.

3 Drag [Grade] to the Columns shelf, to the left of [Loan/Income].

4 Create a Boolean calculation to evaluate whether or not members have both [Dti] < 20 and [Loan/Income] < 15%. Name this calculation "KPI - Low Debt/Small Loan."
Once created, drag this field onto the Color Marks card.

[Loan/Income] <= 0.15 AND [Dti] <= 20

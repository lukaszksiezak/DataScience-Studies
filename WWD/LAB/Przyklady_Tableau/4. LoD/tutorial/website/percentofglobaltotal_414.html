Percent of Global Total
Create a percent of total (seen on the right) that ignores the dimension filter:

1 Start by double-clicking into the second measure on Columns and adding curly braces { } around the measure to return the global total Number of Reviews


{ SUM ( [Number of Reviews] ) }

2 Create a new calculation called "Percent of Global Reviews" using SUM([Number of Reviews]) divided by the newly created global total.


SUM([Number of Reviews]) / SUM({SUM([Number of Reviews])})

3 (Optional) Set the Number Format in default properties of [Percent of Global Reviews] to be a percentage

4 Drag [Percent of Global Reviews] onto Columns and test out the filter

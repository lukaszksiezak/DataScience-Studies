Avg Score per Zip
Find a way to get the average restaurant score per zip code. 
1 Start by creating a calculation that averages the average score to the fixed zip code. Name it "Average Score by Zip".

AVG({FIXED [Restaurant Name] : AVG([Score]) })

2 Drag the new field "Average Score by Zip" to the color shelf

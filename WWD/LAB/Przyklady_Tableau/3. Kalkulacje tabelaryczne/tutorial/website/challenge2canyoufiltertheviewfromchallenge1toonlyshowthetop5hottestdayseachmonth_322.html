Challenge 2: Can you filter the view from Challenge 1 to only show the Top 5 hottest days each month
Now that you have a correctly sorted view, showing the Days in each Month sorted by their Max Temperature (F), can you filed the view to show the Top 5 hottest days in each Month?

1 Recreate the view from Challenge 1, which shows the correctly sorted hottest Days per Month.
SUM(Max Temperature F) should be on the Columns Shelf, with MONTH(Date), Max Temp Rank, and DAY(Date) on the Rows Shelf in that order. Remember to have the Rank Pill compute Pane (down), so that the Rank restarts every Month.

2 Materialize/Save the Rank Calculation we created by dragging the Rank Pill to the Measures Shelf.
Name the field "Rank of Max Temperature"

3 This changes our Rank to a Continuous Measure. Change it back to a Discrete Measure, and place it between MONTH(Date) and DAY(Date).

4 Right-click the "Rank of Max Temperature" Pill, and select "Filter"

5 Select values 1-5 from the pop-up window.

6 Now we have the Top 5 Hottest Days!

7 BONUS: We want to have a slider bar for the end user to interact with, allowing them to change how many Top N Days are visible.

8 Change the Filter to a Continuous Value, to allow for a sliding bar.

9 Change the upper limit to 5, and press OK.

10 Show the Filter. Right-click the "Rank of Max Temperature" on the Filters Shelf, and select "Show Filter"

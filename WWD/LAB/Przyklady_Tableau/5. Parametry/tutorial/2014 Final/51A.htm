5.1: Filter, swap sheets – Fitness challenge (solution)

 

1.      Using the Diabetes rates data source, double-click the State dimension and drag the 18-44 Years Old measure to the color card.  Name the worksheet Diabetes by State.

 

2.      Create a new worksheet.  Using the Exercise calories data source, drag the Calories per hour (130 lbs) measure to the columns shelf and drag the Activity dimension to the rows shelf.  Sort descending.  Name the worksheet Exercise Calories 2.

 

3.      Create a new worksheet.  Using the Food calories data source, drag the Calories measure to the columns shelf and drag the Food dimension to the rows shelf.  Sort descending.  Click the color card and change the color of the bars to orange.  Name the worksheet Food Calories 2.

 

4.      Create a new dashboard and name it Filter, swap sheets 2.  Drag a vertical layout container into the view.

 

5.      Place the Diabetes by State worksheet, the Exercise Calories 2 worksheet, and the Food Calories 2 worksheet in the vertical layout container.

 

 

 

 

6.      On the Diabetes by State worksheet, Analysis > Create Calculated Field…

 

7.      Create a parameter.

 

8.      Name the parameter Choose sheet 2.  Set the data type to integer, the allowable values to list, and type 1, 2, and 3 for the list of values.  Click OK.

 

9.      Write the following calculated field.  Name it Swap sheets 2.

 

10.  Drag the Swap sheets 2 measure to the dimensions pane.

 

11.  Drag the Swap sheets 2 dimension onto the filters shelf.

 

12.  Select custom value list.  Enter 1 and click the + symbol.  Enter 2 and click the + symbol.  Click OK.

 

13.  Right-click the Swap sheets 2 dimension > Copy.

 

14.  On the Exercise Calories 2 worksheet, right-click the dimensions pane > Paste.

 

15.  Drag the Swap sheets 2 dimension onto the filters shelf.  Select custom value list.  Enter 2 and click the + symbol.  Enter 3 and click the + symbol.  Click OK.

 

16.  Right-click the Swap sheets 2 dimension > Copy.

 

17.  On the Food Calories 2 worksheet, right-click the dimensions pane > Paste.

 

18.  Drag the Swap sheets 2 dimension to the filter shelf.  Select custom value list.  Enter 1 and click the + symbol.  Enter 3 and click the + symbol.  Click OK.

 

19.  On the Filter, swap sheets 2 dashboard, click the drop down caret on any of the worksheets > Parameters > Choose sheet 2.

 

20.  For each sheet in the Filter, swap sheets 2 dashboard, right-click the title > Hide Title.

 

 

 

21.  On the Diabetes by State worksheet, right-click the State pill on the marks card > Filter…

 

22.  Select the Top tab and the Top N parameter.  Click OK.

 

23.  On the Exercise Calories 2 worksheet, right-click the Activity pill on the rows shelf > Filter…

 

24.  Select the Top tab and the Top N parameter.  Click OK.

 

25.  On the Food Calories 2 worksheet, right-click the Food pill on the rows shelf > Filter…

 

26.  Select the Top tab and the Top N parameter.  Click OK.

 

27.  On the Filter, swap sheets 2 dashboard, click the drop down caret from any worksheet > Parameters > Top N.

 

28.  Finished.

Concept: We want the ability to filter based on multiple conditions. For example, if we dragged out [Room Type] to the Filter shelf, and selected ‘Private room’ and ‘Shared room’, then all Hosts that have private room’s, shared room’s or both would be returned. In order to only show Hosts that meet both of these conditions, let’s build in some logic to our filter.

 

1.    Right-click and drag ID onto the Text shelf on the Marks card and select Count, CNT(ID).

2.    Ctrl-click CNT(ID) and bring it to the Color shelf. Change the Mark type (from the drop-down list) to Square.

3.    Bring Room Type to the Columns shelf.

4.    Bring Host ID to the Rows shelf.

 

           

 

5.    Bring out Host ID from the Dimensions pane onto Filters.

 

 

6.    Navigate to the Condition tab and select ‘By formula.’ Enter in the formula and press ‘OK’:

MAX(IIF([Room Type] = "Private room",1,0))

+

MAX(IIF([Room Type] = "Shared room",1,0))

= 2

 

*Pay special attention to Room Type’s case sensitivity

 

IIF – or If and Only If – first looks at a condition and checks whether it is true or not. If it is true it returns the first result, else it returns the second result. In the first line of our example, we are returning a value of one if the Room Type is a ‘Private room’, else it will return a zero.

 

Since we are wrapping this expression in MAX(), this expression will either return a 1 or a 0. If we wrapped this same expression in SUM(), it would return a count of how many Private Room’s a particular host is renting out (e.g. if a host has 5 Private Rooms to rent out, it would return 5). But since we are taking the MAX(), no matter how many Private Rooms they rent out, it will still return at most 1.

 

This logic is repeated in the second portion of the calculation for Shared Rooms. Lastly, the two expressions are added together, and only if it is equal to 2 the Host ID is returned. This means that a Host would need to have a Private Room listed (max value of 1) AND a Shared Room listed (max value of 1) to have this condition equal 2, and thus be returned.

 

 

7.    Lastly, select the Quick Sort ‘Ascending’ button to create the final view.

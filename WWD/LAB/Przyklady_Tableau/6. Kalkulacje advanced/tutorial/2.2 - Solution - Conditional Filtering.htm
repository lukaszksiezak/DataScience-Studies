Concept: A conditional filter can be used to filter Host names that were reviewed in a specified year and month. This will not filter out years, therefore allowing us to hold the integrity of the time series. Once we have our Hosts filtered to the subset we are looking for, we can display their review trends from 2014.

 

1.    Right-click under your list of measures and select Create Parameter…

 

 

2.    Name the parameter Year and change the Data Type to Integer. Select List and enter the following values: 2011, 2012, 2013, 2014, and 2015.

 

Be sure to utilize ‘Display As’ to get rid of the comma so the parameter options represent a year value.

 

 

3.    Create another parameter called Month and change the Data Type to Integer. Select Range and enter a Minimum of 1 and a Maximum of 12 to represent the 12 months in a year.

 

 

4.    Bring Host Name to the Filter shelf and navigate to the Condition tab.

5.    Enter in the following formula and select ‘OK’:

MAX(IF YEAR([Review Date]) = [Year] AND MONTH([Review Date]) = [Month] THEN 1 ELSE 0 END) = 1

 

Once again we are using numeric logic to see if a test has passed or failed. In this example, we are checking that the year of a host’s Review Date matches the parameter Year. We then check that the month of a host’s Review Date matches the parameter Month. Only if BOTH of these equations are true (accomplished here by using AND) we return 1. If this condition is equal to 1, then the host is returned, thus allowing us to see the timeline of the host’s reviews in the rest of the data set.

 

This is important, because if we simply filtered on a specified year and month by bringing our Review Date to the filter shelf, we would be limited in seeing the host’s review trends for any time period outside of the one we are filtering on. If the parameters are set to April 2012, I can still see the Review Date trends of those hosts during 2014, which is why a conditional filter is crucial for this problem.

 

 

6.    Bring Review Date to the Filter shelf and select 2014.

 

 

 

Now we can see the time series of reviews for 2014 from the hosts that received reviews in the selected month and year.

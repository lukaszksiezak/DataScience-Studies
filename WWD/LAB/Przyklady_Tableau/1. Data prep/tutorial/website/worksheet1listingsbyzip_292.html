Worksheet 1: Listings by Zip
After using the Data Interpreter, go to Sheet 1 and create this view using Zipcode, Average Price, and Count Distinct of ID:

1 Create a map of Regions
Prepare your data by creating Groups, assigning aliases to the Grouped fields, and renaming the Group to 'Region.'

2 Add labels to the Regions
Add Region, Average Price, and Count Distinct ID to Label. Click Label to customize further. Remember: You can assign the default number format for Price by right clicking the field and modifying the Default Properties.

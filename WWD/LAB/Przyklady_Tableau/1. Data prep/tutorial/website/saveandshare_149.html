Save and Share
We now have a nicely cleaned and prepared data set! Let's make it available for others to use by publishing it.
1 Saving and Sharing your Data Source using Tableau Server
  • Right click the data source, select Publish to Server (You can also publish from the Server menu)
  • Name your data source and set a description (optional)
  • If you want your workbook to use the Published Data Source, check the box at the bottom
  • Click Publish

    • 2 How will other access my data set?
    Now other users may access your data. From Desktop:
  • Select the Tableau Server data source
  • Select your data source

    • 3 (Optional) Saving and Sharing your Data Source using a TDS File
    If you don't have a Tableau Server, you can still save and share your metadata by saving a TDS or TDSX file:
  • Right click the data source and select Add to Saved Data Sources
  • Name the Tableau Data Source
  • Select your file type (TDS will save just the metadata and is appropriate for a live connection. TDSX will save the metadata and the extract together--select this if you want to save and share the metadata and the raw data)

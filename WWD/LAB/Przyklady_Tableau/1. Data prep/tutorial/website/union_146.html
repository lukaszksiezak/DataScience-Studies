Union

You can union your data to combine two or more tables from Excel workbook or text file data. The workbooks or text files must be in the same folder. You union tables by appending values (rows) from one table to another.]


For best results, the tables that you combine must have the same structure. That is, each table must have the same number of fields, and related fields must have matching field names and data types.


In this example, we'll union together several files, each one containing one year's worth of Reviews data for Austin AirBNBs. 



You may notice that two of the columns contain similar data, though in Tableau, they are separated out into two different columns. This is because they have different column header names (Review_ID and ID). Use the "Merge Mismatched Fields" menu option to combine these columns into one for more seamless analysis. 


1 Create a union for Review years 2009-2014
Drag/drop tables for the rest of years (2010-2014) under the AustinAirBNBReviews-2009 file to union them

2 Merge Mismatched Fields
After the union, you may notice that something is wrong with our count of Reviews in our worksheet--the count did not increase. It turns out out in some of the tables, 'Review Id' is named 'id'. We need to merge these columns together so we can count them together in our analysis.

3 Merge columns that have similar data but different column header names
Ctrl + Click on each of the mismatched fields--titled "Review ID" and "ID" from the AustinAirBNBReviews-XXXX data sources. Right click on one field and select “Merge Mismatched Fields.” Rename the new field to "Review ID Number"

4 After merging, replace the old count distinct field with the correct, merged field
Tip: Use a right-click drag to quickly select Count Distinct

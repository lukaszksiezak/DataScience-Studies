Field Names and Properties
Throughout the session we have been adding data through joins and unions, merging fields, renaming fields, and creating calculations. 

All of these steps have been taken us closer to a final model of our data that is specially prepared for analysis. We can share this data model with others, but first let's make some final modifications to the metadata model.

Let's hide some unused fields, rename others, and set some default properties.


1 Hide unused fields from any sheet

  • Return to any of our sheets
  • Hide all unused fields

    By hiding fields, we can disallow our analysts from using certain fields in the data set. This might be for security reasons, or because those fields aren’t useful to our analysts.

    There are a few fields in this data that our analysts won’t need to use--like Lat, Long and Month-Year. These fields weren’t automatically hidden because they are being used in calculations. Remember: Earlier, we created better versions of these fields that should be used in the analysis.
  • Right click and Hide lat, long and Month-Year fields now


    • 2 Create folders
    Since we prepared the data, we know which fields came from the AirBNB data and which fields came from the Median Home Price data. It’s important that our analyst knows this so they don’t mistakenly try to view Median Home Price by Neighbourhood. Remember the Median Home Price data is by zipcode, not neighbourhood.

  • Right click in the white space, Group by Folder
  • Create a folder, name it AirBnB
  • Add the AirBNB related fields to the folder
  • Create a folder, name it Median Home Prices
  • Add the Median Home Prices fields to the folder
  • Repeat this for the measures (or dimensions, if you just did the measures)


    • 3 Rename fields and set default properties

    Now, let’s rename some of these fields:
  • Rename Price to Listing Rate
  • Rename id to Listing ID
  • Rename review_scores_rating to Review Score Rating

    We have some fields that represent dollar amounts and should always be shown as a Currency format.
  • Set default number format properties for Listing Rating and Median Home Price to Currency. Right click --> Default Properties --> Number Format

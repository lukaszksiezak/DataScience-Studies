Worksheet 2: Listings at Lat Long
Now that we have a Latitude and Longitude, we can visualize each listing location on a map. Try to create this map using Latitude, Longitude, ID, and color by Average Review Score Rating.

1 Create the map and color IDs by Rating
• Bring Latitude and Longitude into the view using right-click drag to get the 'Drop Field' pop-up. This shortcut allows me to quickly specify how I want to aggregate the field.
• Bring “id” to the Detail shelf on the Marks card, exclude the data point in Spain
• Change the mark size to a bit smaller from the Size on the Marks card
• Right click drag “review_scores_rating” to the Color shelf, select AVG as the aggregation
• Double click the color legend that appears to change the color palette > Select Red Blue Diverging, select OK
• Add the following Map Layers: Streets and Highways, Place Names

Worksheet 3: Total Reviews
Now that we have our Reviews data, we can create a chart showing the total count of reviews. Create this chart showing a distinct count of review IDs by Year of Date. 

Note: We will only see data from 2009 because the data from other years are in separate tables.


1 Total reviews by year
• Change “Review ID Number” from a Measure to a Dimension
• Bring “Review ID Number” to the Rows shelf > aggregate to Count Distinct (CNTD)
• Bring Date from the Dimensions pane to the Columns shelf
• Change the mark type to Bar from the Marks card
• Exclude Null from the view
• Turn on Labels by selecting the [T] from the top menu bar Since we only have 2009 data, many of the listings we select will show ZERO for Total Reviews. We need to add the reviews from 2010-2015... continue to the next section to find out how to Union data.

Worksheet 4: Dateparse
Our pivoted data now has a usable date field--however, it is being recognized as a string (Abc) field where it should be a date type field. 



We cannot simply convert the field type to a Date type because the format of this date is Month-Year (e.g., Apr-12). 

Let's use a dateparse function to convert the string to a date then build the chart below.


1 Create calculated field
  • First, convert the Data Type of both zipcode fields to a String
  • Next, create the calculation
  • The first parameter of the function should be the format of your date field ("Apr-12" is "MMM-yy")


    • dateparse("MMM-yy", [Month-Year])

    2 With the Month-Year now converted to a date field. We can visualize the Median Prices over time for each zipcode.
  • Add the Median Home Price Date field to Columns, set to Month (continuous)
  • Add Median Home Price to Rows, aggregate to Minimum
  • Add Zipcode to Detail
  • Add Number of Records to Size and Color
  • Set the color and adjust the size

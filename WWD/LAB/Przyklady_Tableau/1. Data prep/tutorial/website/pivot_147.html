Pivot
Sometimes, analyzing data that is stored in a crosstab format can be difficult in Tableau. When working with this type of data, you can pivot your data from crosstab format into columnar format.

Our Austin Median Home Prices.xlsx datasource isn't in an ideal format to analyze in Tableau, so we'll need to do some additional data prep here. In the original datasource, you'll notice that each Month-Year has its own column of data. We'll need to pivot this data for a more seamless analysis--luckily, we can do that right in Tableau! 


Use this time to:
  • Join in the new Excel files for the median home prices
  • Pivot the monthly data in the metadata grid
  • Change Pivot Field Names to "Month-Year"
  • Change the Pivot Field Values to "Median Home Price by Zip"


1 Join the Median Home Price data to the AirBnB Data
  • Click "Add"
  • Select "Austin Median Home Prices.xlsx"
  • Select host_id and Apr-12 as the join fields--these will be our temporary join fields
  • Find the zipcode field in the Median Home Price table--use the color coding--and change the Data Type from Number to String
  • Now, change the join fields to "Zipcode" and "Zipcode, Select Inner join

    • 2 Pivot the Median Home Price Data
  • Select the View Metadata grid icon
  • Scroll down to the bottom of the list to get to the third data sources' fields
  • Select all of the months by Ctrl + Shift + Click > Right click, select "Pivot"
  • Change "Pivot Field Names" to "Month-Year" and "Pivot Field Values" to "Median Home Price"
